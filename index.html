Tesla Fleet Public Key

This service hosts the public key for Tesla Fleet API integration hosted on Nabu Casa Server.

Public Key

Download the public key: tesla_fleet_public_key.pem

Usage

To use this public key with the Tesla Fleet API:

  1. Download the public key file
  2. Configure your Tesla Fleet API integration to use this key
  3. Follow the Tesla Fleet API documentation for further instructions

About

This service is provided by Nabu Casa for Home Assistant Tesla integration.
