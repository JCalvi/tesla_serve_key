Tesla Serve Key — Home Assistant Integration

This integration serves your Tesla Fleet API public key from Home Assistant at the required well-known path: /.well-known/appspecific/com.tesla.3p.public-key.pem

Features

  • Serves Tesla public key at the fixed well-known path required by Tesla Fleet API
  • Unauthenticated endpoint (no authentication required)
  • Automatically searches for PEM file in multiple locations
  • Clear error logging if PEM file is not found
  • No configuration required beyond placing the PEM file

Quick Setup

Step 1: Install the Integration

Via HACS (Recommended):

  1. Add this repository to HACS as a custom repository
  2. Install "Tesla Serve Key" via HACS

Manual Installation:

  1. Copy the tesla_serve_key folder from this repository
  2. Place it in your Home Assistant /config/custom_components/ directory

Step 2: Place Your Public Key

Place your Tesla Fleet API public key in the Home Assistant /config directory with one of these filenames (checked in priority order):

  • tesla-public-key.pem (recommended)
  • tesla_fleet_public_key.pem
  • tesla-fleet-public-key.pem

Step 3: Enable the Integration

Add the following to your configuration.yaml:

tesla_serve_key:

Step 4: Restart Home Assistant

Restart Home Assistant to load the integration.

Step 5: Verify

Confirm the key is being served by visiting: https://yourdomain.tld/.well-known/appspecific/com.tesla.3p.public-key.pem

Important: The integration does NOT include the PEM file. You must provide your own Tesla Fleet API public key file.

How It Works

The integration:

  1. Searches for your Tesla public key in the Home Assistant config directory
  2. Registers a static path handler that serves the file at the required well-known path
  3. Ensures the endpoint is unauthenticated (accessible without Home Assistant login)

Security Notes

  • The endpoint is intentionally unauthenticated as required by Tesla Fleet API
  • Only serve your public key (never include your private key)
  • The well-known path is fixed and cannot be changed (Tesla requirement)

Troubleshooting

Integration fails to load:

  • Check Home Assistant logs for error messages
  • Ensure the PEM file exists in one of the supported locations
  • Verify the PEM file has correct permissions

Key not accessible:

  • Verify Home Assistant is accessible via HTTPS
  • Check for conflicts with other integrations
  • Test from outside your network if Tesla servers need access
